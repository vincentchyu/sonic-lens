# 音眸: UI与视觉整体升级特性清单

## 1. 特性概述
音眸项目在仪表板组件 (`dashboard.html`) 和 Cloudflare Workers 环境前端展示 (`index.html`) 以及全站基础视觉标识进行了美学升级。这些升级引入了现代化动态与统一色彩管理，致力于实现更吸睛且专业的艺术表现风格。

## 2. 核心功能点
1. **仪表板视觉重构**：引入统一色盘映射规则（蓝紫色渐变），不仅覆盖背景和交互控件，还将先前相对静态的流星追踪（"trail"）效果与新背景体系整合；升级网格渐变/丝绸渐变背景替代原先单调背景，并同时保障光/暗双主题功能正常。
2. **Cloudflare 视图同步**：将引入的动态极光背景、Glassmorphism(毛玻璃拟态) 效果以及浮动边栏导航直接平移到 Cloudflare Web 项目主页。
3. **标识（Logo）适配整合**：
   - 动态水印：页面的右下角增加 `logo_all_black.svg` 和 `logo_all.svg` 水印，依照深浅主题自动适配。
   - Favicon 改进：加载 `logo.svg` 和 `logo_black.svg` 以动态映射当前系统模式的主题。

## 3. 设计考量
- **视觉冲击与性能平衡**：增加动态绘制渐变背景和 SVG 图案以吸引用户的同时，在代码层面对元素动画实行足够程度的优化以确保页面平滑不掉帧；
- **全站统一性**：保证不管是主仪表盘还是轻量独立的 Cloudflare 页面都能给出几乎一致的视觉观感。

## 4. 后续演进点
- 继续跟进调整渐变的过渡曲线与颜色以提高文本层次感；
- 扩展不同风格动态背景以供用户在配置选项中手动挑选。
